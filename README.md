# small-business-analysis
Beginner business analysis
## Project Overview
This project analyzes a common small business problem and suggests improvements.

## Business Problem
Customers experience long waiting times and slow responses.

## Stakeholders
- Customers
- Business owner
- Employees

## Current Process (AS-IS)
1. Customer contacts the business
2. Message is received late
3. Response is delayed
4. Customer becomes frustrated

## Proposed Improvements (TO-BE)
- Use one main communication channel
- Set response time goals
- Assign one person to manage messages

## Expected Benefits
- Faster responses
- Happier customers
- Better organization
